# nbp2 中文命名 转 英文命名 拼音法

> naming after pinyin

## 动机

在项目中我们经常要把中文名转换成 `路由名`，`变量名`，`文件名`，`文件夹名` 等等

## 使用

-   帮助文档

```shell
npx nbp2 help
```

-   运行

```shell
npx nbp2 run
```

## flags

-   style

```shell
npx nbp2 run -s camelCase
```

-   prefix

```shell
npx nbp2 run --prefix xxx
```

-   suffix

```shell
npx nbp2 run --suffix xxx
```
